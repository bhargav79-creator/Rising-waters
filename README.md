# 🌊 Rising Waters – Flood Prediction System

## 📌 Project Overview

Rising Waters is an AI-powered flood prediction web application that predicts the likelihood of flooding based on rainfall-related parameters. The project helps users and authorities analyze rainfall data and receive instant flood risk predictions through an easy-to-use web interface.

---

## 🚀 Features

* 🌧️ Predicts flood risk using a Machine Learning model
* 📊 User-friendly web interface
* ⚡ Instant prediction results
* 🔍 Input validation for rainfall parameters
* 📈 Supports disaster preparedness and early warning

## ⚙️ Installation

### Clone the repository

```bash
git clone (https://github.com/k-chandravardhanreddy/Rising_waters)
```

### Navigate to the project folder

```bash
cd Rising_Waters
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python app.py
```

### Open your browser

```text
http://127.0.0.1:5000
```
